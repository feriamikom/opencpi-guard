# Package manifest

This complete package contains:

- `opencpi_guard/`: reusable Python framework code.
- `scripts/`: command-line tools for schema validation, Indonesia conversion, and generic pipeline execution.
- `data/`: schema template, Indonesia reference dataset, and Indonesia common-schema CSV.
- `notebooks/`: Colab GPU local-first notebook package for paper-grade experiments.
- `results/`: hardcore v2.1 model outputs and result summary.
- `submission_easr/`: EASR-ready manuscript pack.
- `docs/`: reproducibility protocol and dataset/manuscript documentation.
- `figures/`: framework diagrams.

Large file note: `results/BPS_CPI_hardcore_v21_outputs.zip` contains trained models, prediction tables, figures, and logs from the paper-grade run.
