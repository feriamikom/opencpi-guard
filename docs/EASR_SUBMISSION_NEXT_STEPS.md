# EASR Submission Next Steps

1. Fill author names and affiliations in the full manuscript.
2. Keep the blind manuscript anonymized.
3. Add funding/acknowledgement if applicable.
4. Create an anonymous GitHub/Zenodo link or state that code/data will be provided upon acceptance.
5. Prepare at least 3 suggested reviewers.
6. Upload the full manuscript, blind manuscript, cover letter, figure captions, and supplementary reproducibility package.
7. Do not overclaim universal data download. Claim country-portable framework via common schema and adapters.
